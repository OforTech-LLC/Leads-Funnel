## Above-Fold Patterns

### Hero Section Structure

1. **Full-width hero image** featuring:
   - Mechanic working on vehicle (action shot)
   - Clean, professional shop environment
   - Customer vehicle being serviced
   - Local shop exterior (for trust)

2. **Headline Formula**
   - Location + Service: "Austin's Trusted Auto Repair Shop"
   - Problem/Solution: "Car Trouble? We Fix It Fast"
   - Value Proposition: "Expert Repairs at Honest Prices"
   - Urgency: "Same-Day Service Available"

3. **Sub-headline Elements**
   - Years in business
   - Certification mention (ASE Certified)
   - Guarantee statement
   - Service area

4. **Above-fold Must-Haves**
   - Phone number (large, clickable)
   - Primary CTA button
   - Trust badges (BBB, ASE, warranties)
   - Hours of operation
   - Address/service area indicator

### Visual Design Patterns

- High-contrast color scheme (often red/blue + white)
- Professional photography over stock images
- Clean, uncluttered layout
- Mobile-first design
- Fast load times (under 3 seconds)

## CTA Patterns

### Primary CTAs

| CTA Text              | Use Case                |
| --------------------- | ----------------------- |
| "Get a Free Quote"    | Lead capture focus      |
| "Schedule Service"    | Appointment booking     |
| "Call Now"            | Immediate contact       |
| "Book Online"         | Self-service scheduling |
| "Request Appointment" | Form-based booking      |

### CTA Placement Strategy

1. **Above the fold** - Primary CTA visible without scrolling
2. **Sticky header** - Phone number + CTA always visible
3. **After each service section** - Contextual CTAs
4. **Bottom of page** - Final conversion opportunity
5. **Exit intent popup** - Discount/coupon offer

### CTA Design Patterns

- Contrasting button color (orange, green, or red)
- Large, thumb-friendly buttons (mobile)
- Action-oriented text (verbs first)
- Urgency indicators ("Today", "Now")
- Secondary CTA for phone calls

## Form Field Patterns

### Minimal Form (Highest Conversion)

```
- Name
- Phone Number
- Service Needed (dropdown)
```

### Standard Quote Form

```
- First Name *
- Last Name
- Phone Number *
- Email
- Vehicle Year/Make/Model
- Service Needed (dropdown/checkbox)
- Preferred Date/Time
- How did you hear about us?
```

### Progressive Disclosure

1. **Step 1**: Contact info (name, phone, email)
2. **Step 2**: Vehicle info (year, make, model)
3. **Step 3**: Service details (issue description)

### Form Best Practices

- Maximum 4-5 fields for initial contact
- Phone number as required field
- Service dropdown with common issues
- Optional "describe your problem" text area
- Clear privacy statement
- Auto-fill support

## Section Structure (Full Page)

### Recommended Section Order

1. **Hero Section**
   - Headline + subheadline
   - CTA button
   - Trust badges
   - Contact info

2. **Services Overview**
   - Grid of service categories with icons
   - Links to service-specific pages
   - "View All Services" CTA

3. **Why Choose Us / Benefits**
   - 3-4 key differentiators
   - Icons + short descriptions
   - Focus on customer benefits

4. **About Section**
   - Brief company story
   - Owner/team photo
   - Local connection emphasis

5. **Testimonials/Reviews**
   - Rotating customer testimonials
   - Star ratings
   - Real customer names/photos

6. **Coupons/Special Offers**
   - Current promotions
   - Printable coupons
   - New customer discount

7. **Service Area**
   - Map integration
   - Neighborhoods served
   - Multiple location info

8. **FAQ Section**
   - Common questions
   - Schema markup for SEO
   - Payment/warranty questions

9. **Final CTA Section**
   - Strong headline
   - Phone number
   - Booking button
   - Hours/address

10. **Footer**
    - Contact details
    - Quick links
    - Social media
    - Privacy/terms

## Objection Handling

### Common Objections & Responses

| Objection                   | Landing Page Solution                               |
| --------------------------- | --------------------------------------------------- |
| "Is this shop trustworthy?" | Certifications, reviews, years in business          |
| "Will they overcharge me?"  | "Free estimates", "No hidden fees", price match     |
| "Is the work quality?"      | Warranty guarantee, ASE certification, before/after |
| "Will it take too long?"    | "Same-day service", "While-you-wait repairs"        |
| "Can I afford it?"          | Financing options, coupons, payment plans           |
| "Do they work on my car?"   | "All makes and models" statement                    |
| "Are they convenient?"      | Hours, location, shuttle service, loaner cars       |

### Objection-Handling Content

- **Warranty Section**: Detailed coverage explanation
- **Pricing Transparency**: "We'll never start work without approval"
- **Expertise Proof**: Tech bios, certifications, training
- **Convenience Features**: Online scheduling, text updates, mobile payments

### Key Conversion Metrics to Target

- Form conversion rate: 5-10%
- Phone call rate: 3-5%
- Average page time: 2+ minutes
- Bounce rate: Under 40%
- Mobile conversion: At least 60% of desktop

### A/B Testing Priorities

1. Headline variations
2. CTA button color/text
3. Form length (3 vs 5 fields)
4. Hero image (mechanic vs shop vs vehicle)
5. Coupon placement
6. Trust badge arrangement
