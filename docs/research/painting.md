## Form Field Patterns

### Quick Quote Form (4-5 fields)

**Essential Fields**

- Name
- Phone number
- Email address
- Service type (Interior, Exterior, Both)
- ZIP code

### Detailed Estimate Form

**Step 1: Project Type**

- Service needed (Interior, Exterior, Both, Commercial)
- Property type (Single family, Townhome, Condo, Commercial)

**Step 2: Project Details**

- Rooms to be painted (for interior)
- Square footage estimate
- Current condition
- Timeline preference

**Step 3: Contact Information**

- Name, phone, email
- Address
- Best time to call

### Form Best Practices

- **4 fields maximum** for initial contact (highest conversion)
- **Service type dropdown** for qualification
- **No-obligation messaging** prominent
- **Mobile-optimized** with large touch targets
- **Quick response promise** ("We'll call within 24 hours")

### Alternative Lead Capture

- Color visualizer tool (lead capture after use)
- Project photo upload
- Callback request
- Live chat

## Section Structure

### Recommended Page Flow

1. **Hero Section**
   - Compelling headline + painted space imagery
   - Free estimate CTA
   - Trust strip (years, homes painted, rating)
   - Phone number

2. **Service Types**
   - Interior Painting
   - Exterior Painting
   - Cabinet Painting
   - Commercial Painting
   - Specialty Services (deck staining, etc.)

3. **Color Visualizer Tool**
   - Upload home photo
   - Try different colors
   - Sherwin-Williams/Benjamin Moore palettes
   - Lead capture integration

4. **Our Process**
   - Free Estimate
   - Color Consultation
   - Surface Preparation
   - Professional Painting
   - Final Walkthrough

5. **Why Choose Us**
   - Professional, insured painters
   - Premium paints used
   - Clean, respectful service
   - Satisfaction guaranteed

6. **Portfolio/Gallery**
   - Before/after photos
   - Interior and exterior projects
   - Filterable by project type

7. **Testimonials**
   - Customer quotes with names
   - Project photos alongside
   - Star ratings
   - Video testimonials

8. **Service Areas**
   - Map or list of areas
   - "Serving [City] and surrounding areas"

9. **FAQ Section**
   - Pricing questions
   - Process questions
   - Timeline questions
   - Paint questions

10. **Final CTA Section**
    - Quote request form
    - Phone number
    - Seasonal special offer

## Objection Handling

### Common Objections & Responses

| Objection                           | Landing Page Response                                                       |
| ----------------------------------- | --------------------------------------------------------------------------- |
| "I can paint it myself"             | Emphasize professional results, time savings, warranty, prep work expertise |
| "Painting is too expensive"         | Show value (longevity, quality), financing options, free estimate           |
| "I'm worried about the mess"        | Clean job site promise, furniture protection, daily cleanup                 |
| "How long will it take?"            | Timeline transparency, "1-day painting" options where applicable            |
| "I don't know what color to choose" | Free color consultation, visualizer tool, expert guidance                   |
| "Will the paint last?"              | Premium paint quality, surface preparation, warranty                        |
| "Bad experiences with painters"     | Reviews, satisfaction guarantee, professional standards                     |

### FAQ Questions to Include

1. How much does house painting cost?
2. How long does a painting project take?
3. Do I need to move my furniture?
4. What kind of paint do you use?
5. Do you offer color consultation?
6. What's included in the estimate?
7. Are your painters insured?
8. What's your warranty?

## Industry-Specific Insights

### What Makes Painting Different

- **Visual transformation** - Before/after critical
- **Color selection anxiety** - Visualizer tools reduce friction
- **DIY comparison** - Need to justify professional value
- **Weather dependent** - Exterior work seasonal

### Color Visualizer Best Practices

- CertaPro's visualizer (My PaintColors) with multiple brand palettes
- Allow home photo upload
- Popular color suggestions
- Lead capture after visualization

### Content That Converts

- Before/after galleries (high engagement)
- Color inspiration content
- "How long will my paint job last?" educational content
- Cost guides by room/project type

### Seasonal Marketing

- Spring/summer: Exterior painting campaigns
- Fall: Interior painting prep for holidays
- Winter: Interior-focused messaging
- Curb appeal timing with real estate seasons

### Mobile Optimization

- Click-to-call essential (60%+ mobile traffic)
- Before/after sliders work well on mobile
- Quick form fills for on-the-go requests
- Color visualizer mobile compatibility
